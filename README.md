# 📊 Excel Dashboard Project

## 📌 Project Overview
This project is an interactive Excel dashboard created to analyze and visualize data in a simple and meaningful way.  
The dashboard helps users understand key business insights by using charts, KPIs, slicers, and data analysis techniques.

## 🎯 Objective
The main objective of this project is to transform raw data into useful insights and present them through an easy-to-understand dashboard.

## 🛠 Tools Used
- Microsoft Excel  
- Power Query  
- Pivot Tables  
- Pivot Charts  
- Formulas and Functions  
- Conditional Formatting  

## 📂 Dataset
The dataset contains information related to **[Sales / Finance / Customer / E-commerce / HR / etc.]**.

Example columns:
- Product Name  
- Category  
- Sales  
- Profit  
- Quantity  
- Region  
- Date  

## 📈 Dashboard Features
- KPI Cards showing important metrics  
- Interactive Slicers for filtering data  
- Bar Chart for category comparison  
- Donut Chart for percentage distribution  
- Monthly Trend Analysis  
- Profit and Revenue Analysis  
- Clean and professional dashboard layout  

## 🔄 Data Cleaning Process
The data was cleaned using Excel and Power Query by:
- Removing duplicates  
- Handling missing values  
- Standardizing date formats  
- Fixing inconsistent text values  
- Creating custom calculated columns  

## 📊 Key Insights
Some insights generated from the dashboard:
- Highest selling category identified  
- Monthly sales trends analyzed  
- Profit margin comparison across products  
- Top performing regions highlighted
